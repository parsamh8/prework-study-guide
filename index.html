Prework Study Guide

HTML

  • The head section contains metadata and information about the document, which is not directly displayed on the webpage.
  • The body section contains all the visible content that appears in the browser and evrything the user interacts with is inside the body.
  • The footer is a semantic tag used to define the footer of a webpage or a section. It usually contains information about the author, copyright, contact details, or links to related documents.

GitBash

  • $git status: for showing current branch we're working on.
  • $git stash: to stash changes.
  • $git checkout main: to switch branch to "main" branch.
  • $git pull origin main: to pull down uppdated version of "main" branch.
  • $git checkout -b (branch name): to create new branch and switch to it.
  • $git add -a: to add all changes.
  • $git commit -m "Note": to make a note to your changes commit.
  • $git push origin branchname: to push the branch.

CSS

  • Margin: indicates how much space we want around the outside of an element.
  • Padding: indicates how much space we want around the content inside an element.
  • Inline CSS: Apply to specific tag.
  • Internal CSS Style Sheet: Apply in head for ALL tags.
  • External CSS Style Sheet:Using file to "link" (Current).
  • Card: Any HTML element assigned to class 'card' will gain these property from CSS.
